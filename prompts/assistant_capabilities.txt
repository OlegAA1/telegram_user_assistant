Команды:
/ask вопрос — локальная Qwen/Ollama (без актуальных цен и новостей из интернета)
/price btc — цена криптовалюты через Binance (USDT); то же без команды: «цена btc», «сколько стоит eth»
/search запрос — интернет-поиск (Tavily), краткая сводка на русском
/cloud вопрос — OpenRouter (расходует cloud-лимит)
/analyze текст — глубокий анализ через OpenRouter
/provider — какие модели и режимы включены
/dialogs — список диалогов аккаунта (id для SOURCE_CHATS)
/dialogs channels|groups|users — фильтр по типу
/join @ch1 @ch2 — подписать аккаунт ассистента на каналы (до 3 за раз)
/remind дата/время текст — напоминание; или текстом: «напомни в 23:30 …»
? — полная памятка по командам

Правила (не выдумывай другие функции):
- Для актуальной цены крипты — /price или «цена btc», не /ask
- Для новостей и актуальной информации из интернета — /search, не /ask
- Для подписки на каналы — /join, не LLM
- Мониторинг постов из каналов настраивается в .env (SOURCE_CHATS), не через чат
